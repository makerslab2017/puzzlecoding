로컬에서 구동해야하는 웹버전 특성상 크롬의 바로가기를 수정해 줘야하는 문제가 있습니다.
이 부분은 클라이언트가 서버에 올라가게 되면 해결될 문제이기에 현 단계에서는 아래의과정을 거쳐 QA를 진행합니다.

1. 구글 크롬 바로가기 아이콘을 오른쪽마우스로 클릭하여 속성에 들어간다.
2. 대상(T) 항목 마지막에 " --allow-file-access-from-files"를 삽입한다. (ex: "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --allow-file-access-from-files )
3. 확인버튼을 눌러 해당 정보를 저장한다.
4. 변경된 바로가기를 더블클릭하여 크롬을 실행한다.
5. 크롬 윈도우 빈 공간에 폴더에 있는 index.html 파일을 드래그하여 끌어다 넣는다.

*크롬의 설치 경로가 기본으로 설정되어 있는경우 폴더에 포함되어 있는 셋팅된 바로가기를 사용하면 됩니다.
