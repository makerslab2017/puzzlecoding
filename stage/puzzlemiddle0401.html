이전에 한번 클리어해봤던 미션이지만 명령어 창의 빈 공간이 줄어든 상태에서 미션을 클리어하기 위해선 다른 방법을 찾아야 해요.

    G1 명령어

적은 수의 명령어를 배치해서 미션을 클리어 하기 위해선 G1 이라는 함수 명령어를 사용해야 해요.
G1 함수 명령어는 똑같은 명령어를 여러 번 반복해서 사용해야 할 경우 G1 명령어 창에 저장해 둔 여러 개의 명령어를 한번에 불러와 사용할 수 있게 해줘요.

    G1 명령어 창

G1 함수 명령어를 사용하기 위해선 G1 명령어 창을 클릭하여 G1 명령어 창에 불이 들어오게 만든 뒤 명령어를 배치해야 해요.

    메인 명령어 창

G1 명령어 창에 명령어 배치가 끝났다면 Main 명령어 창을 클릭해서 불이 들어오게 만든 뒤 G1 명령어를 세개 배치하고, 플레이 버튼을 눌러보세요.
